# SsupMind
This bot is a gentle mental wellness companion designed for daily check-ins.  You can share how you’re feeling in your own words, and the bot will respond with supportive, non-clinical messages to help you pause, reflect, and feel heard. It also helps you build a simple daily check-in streak, encouraging consistency without pressure.
